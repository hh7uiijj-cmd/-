คำนวณ — เลือกวิธีที่ต้องการ
  • ต้องการสถิติรวมทั้งชุด → กด ▶ คำนวณ
  • ต้องการแยกตามกลุ่ม (เพศ/อายุ ฯลฯ) → กด 📋 แยกตามกลุ่ม
✅ ▶ คำนวณ → ตารางแสดง Mean, SD, Min, Max, Median ของทุกตัวแปรที่เลือก
✅ 📋 แยกตามกลุ่ม → แสดง Mean/SD แยกทุก combination ของคอลัมน์หมวดหมู่อัตโนมัติ — รวมแถว "ทุกกลุ่ม"

Z-Score / T-Score / IQ Score
แปลงคะแนนดิบ · หาคะแนนจาก Z · เปรียบเทียบสองตัวแปร
1
ไปที่แท็บ Z/T/IQ

กด 🔬 วิเคราะห์ → 📐 Z/T/IQ

2
เลือกโหมดให้ตรงกับโจทย์
โจทย์ถามว่า…ใช้โหมด
แปลงข้อมูลทั้งชุดเป็น Z-scoreโหมด 1 — เลือกคอลัมน์ → ▶ คำนวณ
X = ค่าใดค่าหนึ่ง → Z = ?โหมด 2 — ป้อน mean, SD และ X
Z → X คืออะไร (ย้อนกลับ)โหมด 2 — สลับป้อน Z แล้วอ่าน X
เปรียบเทียบ Z ของสองตัวแปรโหมด 3 — เลือก X และ Y แล้วกด ▶ ตารางเปรียบเทียบ
3
โหมด 3 — ตารางเปรียบเทียบ

สร้างตาราง N แถว แสดง: X ดิบ / Z(X) / Y ดิบ / Z(Y) / ผลต่าง Z

เลือก ตัวแปร X และ ตัวแปร Y จากรายการ → ▶ ตารางเปรียบเทียบ

✅ Z แต่ละตัวคำนวณจาก mean/SD ของตัวแปรนั้นๆ เอง — สามารถเปรียบเทียบข้ามตัวแปรได้โดยตรง
💡 T-Score = 10×Z + 50  |  IQ-Score = 15×Z + 100 — StatPad คำนวณให้อัตโนมัติในตาราง
3
สหสัมพันธ์ (Correlation)
Pearson r — พร้อม show-work Σ และตารางตีความ
1
เปิดแท็บ สหสัมพันธ์

กด 🔬 วิเคราะห์ → 🔗 สหสัมพันธ์

2
เลือกวิธีและตัวแปร
  • วิธี → Pearson (ข้อมูลต่อเนื่อง/ปกติ) หรือ Spearman (ข้อมูล ordinal/ไม่ปกติ)
  • เลือกตัวแปร → คลิกชื่อคอลัมน์ตัวเลขที่ต้องการ (เลือกได้หลายตัว)
3
กด "คำนวณ" → อ่านค่า r

คลิก ▶ คำนวณ

✅ ตาราง correlation matrix — อ่านค่า r ที่ช่องตัดระหว่างคู่ตัวแปรที่สนใจ
✅ เมื่อเลือก 2 ตัวแปร → กล่อง 📐 แสดงวิธีคิด จะปรากฏด้านล่าง
แสดง x̄, ȳ, Σ(x−x̄)², Σ(y−ȳ)², Σ(x−x̄)(y−ȳ) และสูตร r ครบถ้วน
4
ตีความค่า r
|r|ความหมาย (พวงรัตน์, 2531)ทิศทาง
≥ .80ความสัมพันธ์ สูงมากr > 0 = ไปทางเดียวกัน
r < 0 = ตรงข้ามกัน
r = 0 = ไม่สัมพันธ์
.60 – .79ความสัมพันธ์ สูง
.40 – .59ความสัมพันธ์ ปานกลาง
.20 – .39ความสัมพันธ์ ต่ำ
< .20ความสัมพันธ์ ต่ำมาก หรือไม่มี
💡 r² = Coefficient of Determination — ตีความว่า X อธิบาย Y ได้กี่ % เช่น r = 0.76 → r² = 0.58 → X อธิบาย Y ได้ 58%
4
Regression (Simple & Multiple)
สมการทำนาย + ทำนายค่า X หรือ Y ทันที
1
เปิดแท็บ Regression

กด 🔬 วิเคราะห์ → 📉 Regression

2
เลือกตัวแปร
  • ตัวแปรตาม (Y) → เลือก 1 คอลัมน์ที่ต้องการ "ทำนาย"
  • ตัวแปรอิสระ (X) → เลือก 1 คอลัมน์ (Simple) หรือหลายคอลัมน์ (Multiple)
💡 ถ้าโจทย์ถามให้ทำนาย Y จาก X → Y คือตัวตาม, X คือตัวอิสระ
ถ้าโจทย์ถามให้ทำนาย X จาก Y → สลับ: วาง Y เป็นตัวอิสระ, X เป็นตัวตาม แล้วคำนวณใหม่
3
กด "คำนวณ" → อ่านสมการ

คลิก ▶ คำนวณ

✅ ตาราง Coefficients:
 • Constant = ค่าตัดแกน a
 • ชื่อตัวแปร X = ความชัน b
สมการ: Y' = b₁X₁ + b₂X₂ + … + a
4
(Simple Regression เท่านั้น) ทำนายค่า

เลื่อนลงมาที่กล่อง 🔮 ทำนายค่า ใต้ผลลัพธ์

  • ช่อง "แทน [ชื่อตัวแปร X] =" → พิมพ์ค่า X ที่ต้องการ → → Y' = ?
  • ช่อง "แทน [ชื่อตัวแปร Y] =" → พิมพ์ค่า Y ที่ต้องการ → → X' = ? (คำนวณย้อนกลับ)
✅ แสดงสมการแทนค่าพร้อมขั้นตอน เช่น Y' = 0.89 × 40 + (−7.30) = 28.30
5
ตรวจสอบความดีของสมการ

อ่านค่าในตาราง Model Summary:

ค่าความหมายเป้าหมาย
R²X อธิบายความแปรปรวนของ Y ได้ %ยิ่งสูงยิ่งดี (≥ .50 ถือว่าดี)
SEEความคลาดเคลื่อนมาตรฐานของการทำนายยิ่งน้อยยิ่งแม่น
b (slope)เมื่อ X เพิ่ม 1 → Y เปลี่ยน b หน่วยดูทิศทาง (+/−)
🔎 ตรวจสอบข้ามสมการ: b₁ × b₂ ≈ r² (เมื่อ b₁ = ความชันสมการ Y∼X, b₂ = ความชันสมการ X∼Y) — ใช้ตรวจว่าคำนวณถูกต้อง

# กลุ่มอายุ เพศ วันแรก Z(วันแรก) วันหลัง Z(วันหลัง) ΔZ

ΔZ = Z(วันแรก) − Z(วันหลัง) · สีน้ำเงิน = วันแรกสูงกว่าค่าเฉลี่ยของตนมากกว่าวันหลัง · สีส้ม = วันหลังดีกว่า (เทียบกับค่าเฉลี่ย)
