🚀 ตั้งค่าระบบกิจการนักศึกษา
ทำตามขั้นตอนด้านล่าง ใช้เวลาประมาณ 10–15 นาที
1 สร้าง Firebase Project

Firebase เป็นบริการ Cloud ฟรีของ Google สำหรับเก็บข้อมูลและจัดการผู้ใช้

  1. คลิกปุ่มด้านล่างเพื่อเปิด Firebase Console (ต้อง Sign in ด้วย Google account)
  2. กด Add project หรือ Create a project
  3. ตั้งชื่อ project เช่น student-council
  4. ปิด Google Analytics ได้ (ไม่จำเป็น) → Create project
  5. รอสักครู่จนเสร็จ → Continue
🔥 เปิด Firebase Console
2 เปิดใช้งาน Authentication
  1. ใน Firebase Console → เมนูซ้าย เลือก Authentication
  2. กด Get started
  3. แท็บ Sign-in method → คลิก Email/Password
  4. เปิด Enable → Save
💡 ระบบจะใช้ Email/Password โดยอีเมลจะเป็น รหัสนักศึกษา@student-council.app (ผู้ใช้จะไม่เห็นอีเมลจริง เห็นแค่รหัสนักศึกษา)
3 สร้าง Firestore Database
  1. เมนูซ้าย → Firestore Database → Create database
  2. เลือก Start in production mode → Next
  3. เลือก Location: asia-southeast1 (Singapore) → Enable
  4. รอจนสร้างเสร็จ
  5. ไปที่แท็บ Rules → ลบทุกอย่างแล้ว paste โค้ดด้านล่าง → Publish
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{uid} { allow read: if request.auth != null && (request.auth.uid == uid || isAdmin()); allow write: if isAdmin(); } match /workLogs/{id} { allow read: if request.auth != null && (resource.data.userId == request.auth.uid || isAdmin()); allow create: if request.auth != null && request.resource.data.userId == request.auth.uid; allow update, delete: if isAdmin(); } match /news/{id} { allow read: if request.auth != null; allow write: if isAdmin(); } match /fcmTokens/{uid} { allow read, write: if request.auth != null && request.auth.uid == uid; } match /notifications/{id} { allow read, write: if isAdmin(); } function isAdmin() { return request.auth != null && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin'; } } }
4 รับ Firebase Config
  1. คลิกไอคอน ⚙️ (Project Settings) ด้านบนซ้ายของ Console
  2. แท็บ General → เลื่อนลงหา Your apps
  3. กดไอคอน </> (Web app)
  4. ตั้งชื่อ app (เช่น web) → Register app
  5. Copy ค่าในส่วน firebaseConfig = { ... }

5 สร้างบัญชีประธาน (Admin)
วิธีที่ 1 (แนะนำ): สร้างผ่าน Firebase Console
  1. ไปที่ Authentication → แท็บ Users → Add user
  2. Email: 6501234567@student-council.app
  3. Password: ตั้งรหัสผ่านของคุณ (อย่างน้อย 6 ตัว)
  4. กด Add user → Copy UID ที่แสดง
  5. ไปที่ Firestore → users collection → Add document
  6. Document ID: วาง UID → เพิ่ม fields ด้านล่าง

Fields ที่ต้องเพิ่มใน Firestore document:

{ "studentId": "6501234567", "name": "ชื่อ-นามสกุลของคุณ", "position": "ประธานกิจการนักศึกษา", "role": "admin", "email": "6501234567@student-council.app" }
6 ใส่ config.js เข้าโฟลเดอร์
  1. เปิดโฟลเดอร์ student-council/js/
  2. แทนที่ไฟล์ config.js ด้วยไฟล์ที่ดาวน์โหลดมาจากขั้นตอน 4
  3. หรือเปิดไฟล์ config.js และแก้ไขค่าด้วยตัวเอง
